# SuperDo

> Version: **v1.00.01**  
> Teil des [WebAppX](https://github.com/praetoriani/WebAppX) Projekts

SuperDo ist eine moderne, selbst-enthaltene Single-File Task-Management WebApp – die Weiterentwicklung von *PSAppRocket-ToDo-Demo*, gebaut mit reinem HTML/CSS/JavaScript ohne externe Abhängigkeiten (außer Google Fonts).

---

## Features

- **Slide-In Seitenmenü** – über den Hamburger-Button ein- und ausblendbar, mit sanftem Slide-Effekt
- **Dashboard-Ansicht** – Ringdiagramm-Fortschritt, Prioritäts-Kacheln und Kategorie-Balkendiagramm
- **Statusleiste (unten)** – kompakte Übersicht mit Prioritäten, Fortschrittsbalken und Erledigungs-Zähler
- Dark/Light Theme Toggle
- Volltextsuche
- Filter nach Priorität, Kategorie und Status
- CSV-Export
- Responsive (Mobile-optimiert mit Overlay-Drawer)
- Zustand wird per `sessionStorage` gespeichert
